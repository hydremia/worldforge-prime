# Worldforge Prime Dev Project Guide
This Project acts as the development and testing sandbox for Worldforge Prime.

### Purpose of This Project
- Draft and refine instruction modules.
- Test schemas, Workers, and logic.
- Prepare final <8k instruction block.
- Store temporary or experimental files.
- NOT used for storing canonical lore (use GitHub).

### Recommended Structure
- /instructions/
- /schemas/
- /test_data/
- /nomina_dev/
- /ops_tests/

### Workflow
1. Develop & test here.
2. Validate behavior.
3. Export finalized files to GitHub.
4. Publish only validated elements to the Prime GPT.
